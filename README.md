# Smart Expense Splitter with Debt Simplification

A modern web application that helps users split group expenses fairly and simplifies debts using the **Minimum Cash Flow Algorithm**. The application runs entirely in the browser using HTML, CSS, and JavaScript with LocalStorage support.

## ✨ Features

- 👥 Add, edit, and delete members
- 💸 Add, edit, and delete expenses
- ⚖️ Equal expense splitting
- 🔄 Debt simplification using the Minimum Cash Flow Algorithm
- 📜 Transaction history
- ✅ Mark transactions as settled
- 📊 Dashboard with expense summary
- 📈 Charts using Chart.js
- 🌙 Dark mode
- 🔍 Search and filter expenses
- 💾 LocalStorage support
- 📥 Import JSON
- 📤 Export JSON
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
- Font Awesome
- LocalStorage

## 🚀 How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Start adding members and expenses.

No installation or backend is required.
